# 🏥 AI Hospital & Disease Prediction System

---

## 📌 Overview

The **AI Hospital & Disease Prediction System** is a smart healthcare web application developed using **Streamlit** and **Machine Learning**. The main purpose of this project is to assist users in identifying possible diseases based on the symptoms they select.

The system uses a **Decision Tree algorithm** to analyze user inputs and predict the most likely disease. Along with prediction, it also provides useful healthcare guidance such as recommended doctors, basic medicines, diet suggestions, and overall health analysis.

This project is designed to demonstrate how **Artificial Intelligence can be applied in the healthcare domain** to support early detection and decision-making. It is simple, interactive, and user-friendly, making it suitable for students and beginners learning AI and web app development.

Additionally, the system includes features like a **BMI calculator**, **severity detection**, and a **health score indicator**, which help users better understand their physical condition.

Overall, this project showcases the integration of **Machine Learning, Data Handling, and Frontend UI** into a practical real-world application.
